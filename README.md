## Setup (takes like 2 minutes)

```
1. Head to ascenly.org and hit Download
2. Get Tampermonkey for Chrome or Firefox
3. Paste the script in
4. Open Edgenuity and the panel shows up automatically
```

> Stuck? Hop in the [Discord](https://discord.gg/edgenuity), there's a whole setup channel.

## What it does

- **Auto Answer** - gets every question right automatically
- **Skip Videos** - no more sitting through 10 minute lectures
- **Auto Advance** - moves to the next activity on its own
- **Auto Writer** - writes essays and short answers for you
- **Assignment Unlocker** - unlocks locked activities
- **EdgeEX Support** - works on the new Edgenuity UI too
